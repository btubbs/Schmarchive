Thanks

Schmarchive uses search.secondlife.com for image thumbnails, and inventory icons from the "Silver" skin of the version 1 Second Life Viewer.

Layout and filtering features are provided by the awesome Isotope plugin for jQuery.

License

Schmarchive is open source and licensed under the GPL.

But if you want to set up a copy of Schmarchive on your own server, for commercial use, you'll need to buy a license for Isotope.

Thumbnails

Textures will automatically use search.secondlife.com thumbnails, except for textures with alpha in them (these aren't thumbnailed by search.secondlife.com).

Or you can set your own thumbnail for any item by putting a texture inside the Schmarchive and naming it like ".thumb:ITEM_NAME" (where you replace ITEM_NAME with the actual name of the item). See the thumbnail for the README file as an example.

Styling

Schmarchive also lets you put your own custom CSS on the page. There's a .css notecard inside the prim that will be served as a CSS stylesheet.

Tech

It's built entirely with LSL, Javascript, HTML and CSS. There is no database or server side code.

JSONP is awesome.
