Me

Schmarchive is the creation of Schmobag Hogfather (aka Brent Tubbs in real life).

Tech

It's built entirely with LSL, Javascript, HTML and CSS. There is no database or server side code.

JSONP is awesome.

Thanks

Schmarchive uses search.secondlife.com for image thumbnails, and inventory icons from the "Silver" skin of the version 1 Second Life Viewer.

Layout and filtering features are provided by the awesome Isotope plugin for jQuery.

License

Schmarchive is open source and licensed under the GPL.

But if you want to set up a copy of Schmarchive on your own server, for commercial use, you'll need to buy a license for Isotope.
