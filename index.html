What Is This?

Schmarchive is a web interface for browsing the contents of Second Life objects.

Click on an item from the list above to see details about it, then click "Get This" to have it delivered to you inworld.

Having Problems?

Schmarchive uses new javascript functions that are only supported in the latest browsers. It's known to work in the Firefox 4 Beta and in Google Chrome. (I prefer Chrome.)

If you're not using one of those browsers, now's a good time to start.

There is no spoon server

No database or server side logic is used in Schmarchive. Everything you see on this page is accomplished with regular HTML, CSS, and Javascript.

Your browser talks directly to the LSL scripts inworld using a technique called JSONP. You can put the Schmarchive LSL scripts into any prim and they'll work.
